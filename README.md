# Customer Churn Analysis Dashboard (Power BI)

##Overview
This project focuses on analyzing customer churn to identify patterns, high-risk customers, and key factors contributing to customer loss. The dashboard provides actionable insights to help improve customer retention strategies.



##Problem Statement
Customer churn is a major challenge for businesses. The goal of this project is to:
- Identify customers who are likely to churn
- Analyze factors affecting churn
- Provide insights to reduce customer loss


##Tools & Technologies
- Power BI  
- Power Query  
- DAX  


##Key KPIs
- Total Customers: 7043  
- Churn Rate: 26.54%  
- Churned Customers: 1869  



##Key Insights
- Customers with **month-to-month contracts** have the highest churn rate  
- Customers with **low tenure (0–6 months)** are more likely to churn  
- A significant portion of customers fall under the **high-risk category**  
- Customers without online services show different churn behavior  
- Churn rate decreases as customer tenure increases  



##Dashboard Features
- Customer segmentation based on churn risk  
- Churn rate analysis by tenure  
- Contract-based churn comparison  
- Monthly charges analysis  
- Interactive filters and visuals
